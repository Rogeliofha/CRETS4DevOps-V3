##  Migration Guide

If you are using V1 and want to migrate to V2:

1. **Clone V2 repository**:
   ```bash
   git clone https://github.com/Rogeliofha/CRETS4DevOps-V2.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Copy your customizations** from V1 to V2 if any

4. **Build and test**:
   ```bash
   npm run build
   npm run package
   ```
